# practice

Project ôn Toán rời rạc theo mục tiêu A+ và bám cấu trúc đề thi năm nay.

## Cách dùng nhanh

Trong mỗi cuộc chat ôn tập, bắt đầu bằng:

```md
Tuân thủ `practice/AGENTS.md`.
```

`AGENTS.md` là luật lõi. AI phải đọc file đó trước, tự chọn rule trong `rules/`, rồi mới trả lời hoặc tạo bài.

## Bản đồ project

- `AGENTS.md`: luật lõi, thứ tự đọc context, vai trò gia sư và điều phối rule.
- `rules/`: workflow cho hỏi lý thuyết, giải bài, lọc bài tập, phân tích đề và tạo đề.
- `EXAM_CONTEXT.md`: cấu trúc đề thi năm nay, nguồn ưu tiên cao nhất khi ôn thi.
- `EXAM_ANALYSIS.md`: nơi phân tích đề cũ, chỉ dùng để tham khảo độ khó.
- `EXAM_PATTERNS.md`: dạng bài luyện tập theo cấu trúc năm nay và độ khó tham khảo.
- `LEARNING_PLAN.md`: thứ tự ôn và ưu tiên theo mục tiêu A+.
- `FORMULAS.md`, `MISTAKES.md`, `QUESTIONS.md`: hồ sơ học tập cá nhân.
- `topics/`: tổng kết lý thuyết theo tuần/chương.
- `exercises/`: bản đồ bài tập ưu tiên theo homework.
- `exams/`: đề cũ, đề luyện và ma trận phân tích.
- `materials/`: PDF nguồn và text trích xuất.
- `figures/`: hình vẽ đồ thị, cây, ôtômát, bảng chuyển trạng thái.
