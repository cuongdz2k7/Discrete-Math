# topics

Thư mục này chứa tổng kết lý thuyết theo tuần/chương.

## Cấu trúc chuẩn mỗi file

- Vai trò trong đề thi.
- Nguồn đã nạp.
- Kiến thức cốt lõi.
- Định nghĩa/công thức/thuật toán cần nhớ.
- Điều kiện áp dụng.
- Dạng bài hay ra.
- Quy trình giải chung.
- Lỗi sai thường gặp.
- Bài đại diện nên xem lại.
- Câu hỏi còn mở.

## Quy tắc

- Không chép lại slide dài dòng.
- Viết theo hướng ôn thi: nhận dạng, công cụ, bẫy, trình bày.
- Với Đồ thị, Cây, Ôtômát hữu hạn, phải ghi rõ phần tự luận.
