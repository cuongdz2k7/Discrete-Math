# exercises

Thư mục này chứa bản đồ bài tập ưu tiên theo homework.

## Cấu trúc chuẩn mỗi file priority

- Vai trò trong đề thi.
- Nguồn bài tập.
- Bảng lọc bài tập.
- Công thức/định nghĩa bắt buộc.
- Khung tư duy theo dạng bài.
- Thứ tự làm bài đề xuất.
- Gợi ý ôn nhanh.
- Lỗi dễ mất điểm.

## Quy tắc

- Priority file không phải lời giải đầy đủ.
- Mỗi bài phải có lý do ưu tiên dựa trên `EXAM_CONTEXT.md` trước.
- Đề cũ chỉ dùng để cân độ khó nếu đã được phân tích.
- Bài A+ phải khó vừa sức thi, không tính toán rườm rà.
