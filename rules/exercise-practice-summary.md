# Rule tổng hợp bài tập homework

Dùng khi người học gửi homework hoặc muốn tạo/cập nhật file `exercises/*-priority.md`.

## File cần đọc

- `AGENTS.md`.
- `EXAM_CONTEXT.md`.
- `EXAM_PATTERNS.md`.
- `EXAM_ANALYSIS.md` nếu cần tham khảo độ khó.
- `LEARNING_PLAN.md`.
- PDF homework hoặc `materials/extracted/homework/*.txt`.
- File `topics/` liên quan.
- File `exercises/*-priority.md` cũ nếu đang cập nhật.

## Việc cần làm

1. Đọc danh sách bài tập.
2. Phân loại từng bài theo chủ đề, dạng, mức độ và kỹ năng kiểm tra.
3. Gắn ưu tiên: bắt buộc, nên làm, làm nếu còn thời gian, bỏ qua.
4. Ghi căn cứ ưu tiên theo `EXAM_CONTEXT.md` trước; đề cũ chỉ dùng để cân độ khó.
5. Tạo bảng công thức/định nghĩa bắt buộc.
6. Tạo khung nhận dạng dạng bài:
   - Nhận dạng trong đề.
   - Công cụ bật ra đầu tiên.
   - Cách chọn bước tiếp theo.
   - Cách trình bày.
   - Bẫy nhận dạng sai.
   - Lỗi dễ mất điểm.
7. Đề xuất thứ tự làm bài từ nền đến A+.

## Khi cập nhật file

- File đích mặc định: `exercises/tuan-xx-<slug>-priority.md`.
- Không ghi lời giải đầy đủ từng bài trong file priority.
- Không cập nhật `EXAM_PATTERNS.md` nếu chưa được phép.
