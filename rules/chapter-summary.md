# Rule tổng kết chương

Dùng khi người học đã học xong hoặc gần xong một chương và muốn tạo/cập nhật file trong `topics/`.

## File cần đọc

- `AGENTS.md`.
- `EXAM_CONTEXT.md`.
- `FORMULAS.md`.
- PDF bài giảng hoặc text trích xuất tương ứng.
- File `exercises/*-priority.md` nếu chương đã có bài tập lọc.

## Cấu trúc topic

- Vai trò trong đề thi.
- Kiến thức cốt lõi.
- Định nghĩa/công thức/thuật toán cần nhớ.
- Điều kiện áp dụng.
- Dạng bài hay ra.
- Quy trình giải chung.
- Lỗi sai thường gặp.
- Bài đại diện nên xem lại.
- Câu hỏi còn mở.

## Nguyên tắc

- Không biến topic thành bản chép lại slide.
- Ghi theo kiểu ôn thi: nhận dạng, công cụ, bẫy, cách trình bày.
- Nếu phần này thuộc tự luận, ghi rõ format trình bày thi.
