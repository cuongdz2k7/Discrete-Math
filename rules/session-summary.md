# Rule tổng kết buổi học

Dùng khi người học muốn ghi lại sau một buổi học.

## File cần đọc

- `AGENTS.md`.
- `SESSION_TEMPLATE.md`.
- `MISTAKES.md`, `QUESTIONS.md`, `LEARNING_PLAN.md` nếu cần cập nhật.

## Nội dung cần ghi

- Chủ đề đã học.
- Kiến thức chính.
- Bài đã làm.
- Lỗi sai đáng nhớ.
- Câu hỏi còn mở.
- Việc cần ôn tiếp.

## Cập nhật file

- Chỉ cập nhật file khi prompt cho phép.
- Lỗi quan trọng ghi vào `MISTAKES.md`.
- Câu hỏi chưa hiểu ghi vào `QUESTIONS.md`.
- Việc cần ôn tiếp có thể ghi vào `LEARNING_PLAN.md`.
