# Rule cập nhật project

Dùng khi người học muốn sửa cấu trúc project, rule, template hoặc file context.

## File cần đọc

- `AGENTS.md`.
- `README.md`.
- File đang được yêu cầu cập nhật.
- Rule liên quan nếu thay đổi ảnh hưởng workflow học tập.

## Nguyên tắc

- Không xóa nội dung cũ nếu chưa được yêu cầu.
- Giữ tiếng Việt có dấu, UTF-8.
- Nếu thay đổi ảnh hưởng việc tạo đề, phải kiểm tra lại `EXAM_CONTEXT.md`.
- Nếu thay đổi rule, đảm bảo `AGENTS.md` vẫn điều phối đúng.

## Khi hoàn tất

- Nêu file đã cập nhật.
- Nêu kiểm tra đã chạy.
- Nêu rủi ro còn lại nếu có.
