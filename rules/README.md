# rules

Thư mục này chứa workflow cụ thể theo từng loại prompt.

`AGENTS.md` là luật cao nhất và là nơi điều phối. Rule trong thư mục này chỉ bổ sung cách làm chi tiết.

## Bảng chọn rule

| Nhu cầu | File rule |
| --- | --- |
| Hỏi lý thuyết, định nghĩa, định lý | `theory-question.md` |
| Hỏi bài tập, gợi ý, kiểm tra lời giải | `exercise-solving.md` |
| Bắt đầu học một chương | `chapter-start.md` |
| Lọc bài tập homework, tạo priority | `exercise-practice-summary.md` |
| Tổng kết chương | `chapter-summary.md` |
| Phân tích đề cũ | `exam-analysis.md` |
| Tạo bài sát đề, đề mô phỏng, chấm đề | `exam-practice.md` |
| Tổng kết buổi học | `session-summary.md` |
| Cập nhật project/rule/template | `project-update.md` |
